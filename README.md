# ✋ Hand Gesture Draw

A Python-based AI project that allows users to draw on screen using hand gestures. Built using OpenCV and MediaPipe for real-time hand tracking.

## 🚀 Features
- Draw using hand gestures
- Real-time tracking
- Smooth drawing experience
- Interactive UI

## 🛠️ Tech Stack
- Python
- OpenCV
- MediaPipe

## ▶️ How to Run
```bash
pip install -r requirements.txt
python main.py
